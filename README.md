# 手写简易版Vuex项目my-vuex3

项目描述：旨在通过手写实现一个简易版的Vuex，帮助深入理解Vuex的工作原理和内部实现机制。项目涵盖了Vuex的核心功能，包括状态管理、Getter、Mutation、Action以及模块化支持。

## Project setup
```
npm install
```

npm run serve
